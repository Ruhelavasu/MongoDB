# MongoDB Learning Journey

This repository documents my MongoDB learning journey through daily practice, notes, queries, schema design, and mini backend projects.

The goal of this repo is to build practical MongoDB skills in public, stay consistent with daily learning, and show visible progress over time.

## Why I am building this

I am creating this repository to:
- Practice MongoDB consistently
- Document what I learn in a structured way
- Build small hands-on examples instead of only reading theory
- Maintain a public learning record on GitHub

## Learning goals

By the end of this repository, I want to be comfortable with:
- MongoDB basics
- CRUD operations
- Query operators
- Schema design
- Indexing
- Aggregation pipeline
- Using MongoDB with Node.js and Express
- Building small backend projects with MongoDB
